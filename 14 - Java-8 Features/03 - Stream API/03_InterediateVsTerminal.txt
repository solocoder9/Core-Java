+-----------------------+-------------------------------------------------+-----------------------------------------------------+
|          Aspect       |            Intermediate Operations              |              Terminal Operations                    |
+-----------------------+-------------------------------------------------+-----------------------------------------------------+
| Effect                | Return a new Stream (lazy)                      | Produce a result or side-effect (eager)             |
+-----------------------+-------------------------------------------------+-----------------------------------------------------+
| Execution             | Lazy: Not executed until terminal operation is  | Eager: Triggers stream processing                   |
|                       | invoked                                         |                                                     |
+-----------------------+-------------------------------------------------+-----------------------------------------------------+
| Reuse                 | Can be chained together, stream remains open    | Stream is consumed after execution, cannot be       |
|                       |                                                 | reused                                              |
+-----------------------+-------------------------------------------------+-----------------------------------------------------+
| Examples              | filter(), map(), flatMap(), distinct(), sorted()| collect(), forEach(), reduce(), count(), anyMatch() |
+-----------------------+-------------------------------------------------+-----------------------------------------------------+
