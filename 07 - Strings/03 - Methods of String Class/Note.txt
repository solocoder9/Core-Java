String:

- String is an inbuilt class which present in java.lang package which is use to 
store group of characters.
- String value internally sotred as characters array.
- String values alsways define with "" double quortes
- String object can be created in following two ways
    1> by using Constructor
       ex - String s = new String("java");

    2> without Constructor (literals)
       ex - String s = "Java"
